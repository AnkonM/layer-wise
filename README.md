# LayerWise

LayerWise is a milestone-oriented scaffold for a three-layer system with a pure Python ML engine, a FastAPI backend, and a React frontend. This repository provides the initial project structure and minimal stubs needed to build features incrementally without restructuring the architecture.

## Setup

### Python

Install runtime and development dependencies.

### API

Run the FastAPI application entrypoint.

### Frontend

Install frontend dependencies and start the Vite development server.

### Tests

Run the test suite from the repository root.
